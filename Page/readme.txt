Partie Html :
- Quelques connaissances grâce aux quelques heures de cours sur HTML en 1ère année de Licence Informatique
  Cours Open Classroom HTML5/CSS3
- La balise 'header' inclu les informations techniques sur la RFC
  Les balises 'section', situées avant et après la balise 'article', contiennent des informations telles que l'abstract, le status, le sommaire, les considérations, les références... Bien que certaines informations ne soient pas forcement présentes dans toutes les RFC
  La balise 'article' contient les informations principales et detaillées de la RFC
  La balise 'footer' rassemble les informations sur les auteurs (adresse, téléphone, email)
  Une balise 'aside' permet d'inclure un 'Full Copyright Statement' si nécessaire

Partie CSS :
- Cours Open Classroom HTML5/CSS3
- Ayant quelques connaissance des Flexbox mais n'étant pas encore très à l'aise avec, j'ai décidé de choisir cet outil CSS pour ce projet afin d'être plus à l'aise et de les utiliser plus facilement
